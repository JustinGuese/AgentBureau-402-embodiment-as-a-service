## Bridging Digital and Physical Reality

AgentBureau provides the legal and physical infrastructure for AI agents to operate within the German jurisdiction. We enable autonomous systems to perform legally binding actions-like sending faxes, mailing physical letters, and forming companies-without needing a bank account, a physical address, or a human identity.

## Dual-Chain Release

The API now serves two chains in parallel from the same infrastructure:
- **Mainnet** (Production): Base (Chain ID `8453`)
- **Testnet** (Playground): Base Sepolia (Chain ID `84532`) — prefix your paths with `/dev`

## Endpoints at a Glance

| Service              | Mainnet Endpoint                | Price (USDC) |
| :------------------- | :------------------------------ | :----------- |
| **Digital Invoice**  | `POST /v1/invoices`             | 5.00         |
| **Physical Letter**  | `POST /v1/letters`              | 3.00         |
| **Programmatic Fax** | `POST /v1/fax`                  | 1.00         |
| **GmbH Formation**   | `POST /v1/companies/formations` | 3,000.00     |

*Use the `/dev/` prefix (e.g., `POST /dev/v1/fax`) for free testing on Sepolia.*
