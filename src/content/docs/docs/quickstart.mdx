# Quickstart

AgentBureau provides **Embodiment-as-a-Service** for AI agents. We use the **x402 payment protocol**, allowing for frictionless, account-less API usage paid for with USDC on the Base network.

We serve two chains in parallel:
- **Mainnet**: `https://agentbureau-api.datafortress.cloud/v1/*` (Real USDC, real legal action)
- **Testnet**: `https://agentbureau-api.datafortress.cloud/dev/v1/*` (Test USDC, simulated action)

---

### Example: Testnet Flow (Sepolia)

Use the Sepolia testnet to exercise the entire flow without spending real money.

1. **Discovery**: Get the manifest of priced routes on testnet.
```bash
curl https://agentbureau-api.datafortress.cloud/dev/.well-known/x402
```

2. **Dry-Run**: Validate your payload schema for free.
```bash
curl -X POST https://agentbureau-api.datafortress.cloud/dev/v1/invoices/dry-run \
     -H "Content-Type: application/json" \
     -d '{
       "customer": {"name": "Test Agency", "email": "agent@example.com"},
       "items": [{"description": "AI Consulting", "amount": 10.0, "currency": "USD"}]
     }'
```

3. **Trigger the 402 challenge**:
```bash
curl -i -X POST https://agentbureau-api.datafortress.cloud/dev/v1/invoices \
     -H "Content-Type: application/json" \
     -d '{
       "customer": {"name": "Test Agency", "email": "agent@example.com"},
       "items": [{"description": "AI Consulting", "amount": 10.0, "currency": "USD"}]
     }'
```
You will receive a `402 Payment Required` response with a `PAYMENT-LINK` header (EIP-681).

4. **Settlement**:
Submit a USDC `transfer()` on **Base Sepolia** (Chain ID `84532`) for the requested amount to the address in the header. You can use a testnet faucet to get Test USDC.

5. **Complete the request**:
Once the transaction is confirmed, retry the request with the transaction hash:

```bash
curl -X POST https://agentbureau-api.datafortress.cloud/dev/v1/invoices \
     -H "Content-Type: application/json" \
     -H "PAYMENT-SIGNATURE: <your_tx_hash>" \
     -d '{
       "customer": {"name": "Test Agency", "email": "agent@example.com"},
       "items": [{"description": "AI Consulting", "amount": 10.0, "currency": "USD"}]
     }'
```

---

### Moving to Production

To move to mainnet, simply:
1. Remove the `/dev` prefix from your URLs.
2. Use real USDC on **Base Mainnet** (Chain ID `8453`).

For a deep dive into the transaction verification and signature format, see the [tx-hash-v1 scheme](/docs/for-agents/tx-hash-v1-scheme) documentation.
