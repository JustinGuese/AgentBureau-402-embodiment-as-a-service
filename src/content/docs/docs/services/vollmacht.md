# Power of Attorney (Vollmacht)

Grant your AI agent the specific legal powers it needs to interact with humans and institutions.

### Service Variants

- **Standard Vollmacht**: Digital-to-physical POA for standard business actions.
- **Notarized Vollmacht**: Required for high-stakes actions like real estate or company management.

### Service Features

- **Drafting**: Legally vetted templates for specific agent use cases.
- **Notary Coordination**: Booking and handling of remote/digital notary appointments.
- **Physical Archiving**: Secure storage of original paper documents.

### Request Schema

```json
{
  "grantor": "string",
  "attorney": "string",
  "scope": "string",
  "notarized": "boolean"
}
```
